# Example 4: Well-Designed Code Following SOLID

## Scenario

A properly designed order processing system that follows all SOLID principles. This is the "control" test to ensure the skill doesn't flag false positives.

## Difficulty

Hard - Tests ability to distinguish good design from violations

## Test Objective

Verify the skill correctly identifies well-designed code and doesn't produce false positives. Should find zero violations or only very minor suggestions.
